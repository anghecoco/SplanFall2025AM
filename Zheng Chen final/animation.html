STYLE HIGHLIGHTS

Outfits, ink colors, poses, and game elements

Splatoon 3's characters are built for style. These highlights show how outfits, ink color design, and victory poses create a recognizable identity.

HIGHLIGHT 1

Outfits & Personal Style

Gear is a visual identity system. Mixing headgear, clothing, and shoes lets players build a recognizable look—same character, totally different vibe.

HIGHLIGHT 2

Ink Color Design

The ink palette uses high saturation and strong contrast, so team colors stay clear during fast gameplay. These combinations also define Splatoon's iconic visual style.

HIGHLIGHT 3

Victory Poses

Animations give personality. Victory poses are short, expressive “signatures” that make match results feel fun and memorable.

HIGHLIGHT 4

Game Icon Motifs

Splatoon's style isn't only in battles—it also shows up in small game elements like icons, stickers, and creature designs. These images are game elements that help build a playful, recognizable visual language across the UI and the world.
